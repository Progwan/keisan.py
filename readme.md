# 使い方
　- pythonがインストールされている必要があります
## 最初にやること
  ```~/.bashrc``` 
または```~/.bash_profile```に
  ```
    export $PATH=計算アプリの入っているフォルダの絶対パス```
  次に
```
    source ~/.bashrc(または~/.bash_profile)
  そうして、
```
  chmod 777 setup.py
  chmod 777 keisan.py
  setup.py
```
とします。
## 使う
```
  keisan.py --help```を参照
